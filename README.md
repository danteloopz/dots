## How to install?

Run this command to install:
```
sudo pacman -Sy --needed curl
curl -sSL https://raw.githubusercontent.com/BinaryHarbinger/hyprdots/refs/heads/main/install.sh -o install.sh
chmod +x install.sh
./install.sh
```
